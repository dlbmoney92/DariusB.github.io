Drug Prediction Project

This project aims to predict the type of drug suitable for patients based on health parameters such as age, gender, blood pressure, and cholesterol levels. The dataset used for this project is sourced from this GitHub repository.

Technologies Used

  • Python: For data analysis and machine learning implementation.
  • Libraries:
    • Pandas and NumPy for data manipulation and analysis.
    • Scikit-learn for machine learning algorithms.
    • Matplotlib and Seaborn for data visualization.

Steps in the Project

  1. Data Cleaning: Handled missing values and encoded categorical variables.
  2. Exploratory Data Analysis (EDA): Visualized relationships between variables to uncover patterns.
  3. Feature Engineering: Selected key features influencing drug prediction.
  4. Model Training: Trained multiple machine learning models, including Logistic Regression, Random Forest, and Support Vector Machine (SVM).
  5. Evaluation: Compared model performance using accuracy, precision, and recall metrics.

Key Insights

  • Blood pressure and cholesterol levels were critical factors in drug classification.
  • Random Forest and Decision Tree models achieved the highest accuracy among tested algorithms.
