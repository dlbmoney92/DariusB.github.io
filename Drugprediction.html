Drug Prediction

Project Overview

This project utilizes machine learning techniques to predict the presence of heart disease in patients based on health parameters such as age, gender, cholesterol levels, and more. The goal is to provide a predictive model that aids in early diagnosis and better treatment planning.

About the Dataset:

The dataset used in this project contains various health-related metrics, including:

  • Age: Patient age in years.
  • Sex: Gender of the patient (1 = male, 0 = female).
  • Chest Pain Type: Type of chest pain experienced (e.g., typical angina, atypical angina).
  • Resting Blood Pressure: Resting blood pressure in mmHg.
  • Cholesterol: Serum cholesterol in mg/dL.
  • Fasting Blood Sugar: Fasting blood sugar > 120 mg/dL (1 = true, 0 = false).
  • Maximum Heart Rate Achieved: Maximum heart rate achieved during exercise.
  • Exercise Induced Angina: Presence of exercise-induced angina (1 = yes, 0 = no).

Steps in the Project

  1. Data Cleaning: Handled missing values and standardized numerical features.
  2. Exploratory Data Analysis (EDA): Visualized the distribution of features and their correlation with heart disease.
  3. Feature Selection: Selected the most relevant features for training the model.
  4. Model Training: Trained multiple machine learning models, including Logistic Regression, Random Forest, and XGBoost.
  5. Evaluation: Compared model performance using metrics such as accuracy, precision, recall, and F1-score.
