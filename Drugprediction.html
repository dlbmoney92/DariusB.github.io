Drug Prediction Project

Project Overview

The Drug Prediction Project involves analyzing patient health parameters to classify the appropriate drug for treatment. Using machine learning models, the project predicts drug categories based on features like age, gender, blood pressure, cholesterol, and more.

About the Dataset

The dataset includes the following attributes:

  • Age: Patient's age in years.
  • Sex: Gender of the patient.
  • Blood Pressure: Categorical representation of blood pressure (Low, Normal, High).
  • Cholesterol: Categorical cholesterol levels (Normal, High).
  • Drug Type: The drug prescribed for treatment.

Steps in the Project

  1. Data Cleaning: Addressed missing values and encoded categorical variables.
  2. Exploratory Data Analysis (EDA): Identified correlations between features and drug type.
  3. Feature Engineering: Created meaningful features for model input.
  4. Model Training: Implemented models such as Logistic Regression, Decision Tree, and Random Forest.
  5. Evaluation: Assessed performance using metrics like accuracy, precision, and F1-score.

Results

The analysis produced the following insights:

  • Decision Tree: Achieved the best performance with an accuracy of 90%.
  • Logistic Regression: Delivered consistent results with an accuracy of 85%.
  • Feature Importance: Blood pressure and cholesterol levels were significant predictors of drug classification.
