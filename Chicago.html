SQL Chicago Project

Project Overview

The SQL Chicago Project involves exploring a public dataset from the city of Chicago to extract insights using SQL queries. The project focuses on analyzing data related to demographics, public amenities, crime statistics, and economic factors to uncover trends and patterns that impact the cityâ€™s residents.

About the Dataset

The dataset contains information from various sectors of the city, including:

  • Demographics: Population distribution across neighborhoods.
  • Public Services: Details about schools, libraries, and parks.
  • Crime Data: Incidents reported by type and location.
  • Business Activity: Insights into local businesses and economic growth.

Steps in the Project

  1. Database Setup: Imported and structured data in a relational database.
  2. SQL Queries: Created and executed queries to analyze various metrics, such as crime rates and business density.
  3. Insights Extraction: Used aggregation, filtering, and joins to identify trends and correlations.
  4. Visualization: Exported query results to create charts and graphs for better interpretation.

Results

The project produced the following insights:

  • Crime Analysis: Identified neighborhoods with the highest crime rates and the types of crimes most commonly reported.
  • Public Services: Correlated the availability of schools and libraries with population density across neighborhoods.
  • Economic Factors: Analyzed the distribution of businesses and economic growth patterns in various areas.
