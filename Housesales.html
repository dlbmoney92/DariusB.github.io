House Sale Analysis Project

Project Overview

The House Sale Analysis Project involves analyzing a dataset containing information about house sales. The goal is to identify key factors affecting house prices and build predictive models to estimate house values based on various features such as location, size, and condition.

About the Dataset

The dataset includes comprehensive information on house sales, such as:

  • Sale Price: The final price of the house.
  • Location: The geographic area of the property.
  • Square Footage: The size of the house in square feet.
  • Number of Rooms: The count of bedrooms and bathrooms.
  • Condition: The overall condition of the house.
  • Year Built: The year the house was constructed.

Steps in the Project

  1. Data Cleaning: Handled missing values, outliers, and inconsistencies in the dataset.
  2. Exploratory Data Analysis (EDA): Visualized trends, distributions, and correlations between features and house prices.
  3. Feature Engineering: Created new features, such as age of the house and price per square foot.
  4. Model Training: Implemented models such as Linear Regression, Random Forest, and Gradient Boosting to predict house prices.
  5. Evaluation: Compared model performance using metrics like Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).

Results

The project yielded the following insights and outcomes:

  • Square Footage: Found to be the most significant predictor of house prices.
  • Condition: Houses in better condition commanded higher prices.
  • Random Forest: Achieved the best performance with an RMSE of $15,000.
  • Gradient Boosting: Provided robust predictions with an RMSE of $16,500.
