SpaceX Capstone Project

Project Overview

The SpaceX Capstone Project is a data science initiative that analyzes SpaceX launch data to predict the success of future launches. The project involves data collection, preprocessing, exploratory data analysis (EDA), and the implementation of machine learning models to identify critical factors influencing launch outcomes.

About the Dataset

The dataset includes details of SpaceX launches, covering variables such as:

  • Launch Site: The location of the launch.
  • Payload Mass: The weight of the payload in kilograms.
  • Orbit: The target orbit of the mission.
  • Booster Version: The type of rocket booster used.
  • Launch Outcome: Success or failure of the launch.

Steps in the Project

  1. Data Collection: Retrieved data from the SpaceX API and additional sources.
  2. Data Cleaning: Processed and standardized data to handle missing or inconsistent values.
  3. EDA: Analyzed trends and relationships between variables such as payload mass and orbit type.
  4. Model Training: Implemented machine learning models including Logistic Regression, Random Forest, and Support Vector Machines (SVM).
  5. Evaluation: Assessed model performance using accuracy, precision, and recall metrics.

Results

The analysis provided the following key insights:

  • Random Forest: Achieved the highest accuracy of 91%, making it the best-performing model for predicting launch success.
  • Logistic Regression: Delivered consistent results with an accuracy of 85%.
  • Payload Mass: Found to be a significant factor influencing the likelihood of launch success.
