Heart Disease Prediction Project

Project Overview

This project uses machine learning techniques to predict the presence of heart disease in patients based on health parameters such as age, gender, cholesterol levels, and blood pressure. The goal is to provide a predictive model that aids in early diagnosis and better treatment planning.

About the Dataset

The dataset includes various health-related metrics, including:

  • Age: Patient's age in years.
  • Sex: Gender of the patient.
  • Blood Pressure: Resting blood pressure level.
  • Cholesterol: Serum cholesterol in mg/dL.
  • Max Heart Rate: Maximum heart rate achieved during exercise.
  • Fasting Blood Sugar: Fasting blood sugar > 120 mg/dL (1 = true, 0 = false).
  • Exercise Induced Angina: Presence of exercise-induced angina.

Steps in the Project

  1. Data Cleaning: Handled missing values and standardized features.
  2. Exploratory Data Analysis (EDA): Analyzed feature distributions and their correlation with heart disease.
  3. Model Training: Implemented Logistic Regression, Random Forest, and XGBoost models.
  4. Evaluation: Compared performance using metrics like accuracy and F1-score.

Results

The analysis provided the following key insights:

  • Logistic Regression: Achieved baseline accuracy of 78%.
  • Random Forest: Delivered the highest accuracy of 88%, making it the best-performing model.
  • XGBoost: Achieved 85% accuracy with robust feature importance analysis.
  • Cholesterol Levels and Max Heart Rate: Identified as the most significant factors influencing heart disease predictions.
